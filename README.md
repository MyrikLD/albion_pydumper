# albion_pydumper
This tool allows you to dump the info in the Albion Online ".bin" Files, convert them to XML/JSON files.

## How to use:
```bash
# Download binary files
python downloader.py

# Decode data from albion folder
python decryptor.py

# Convert xml to json
python xml_to_json.py

# Or just run
python main.py
# to get json
```
